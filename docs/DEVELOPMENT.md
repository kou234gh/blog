# 開発ガイド

ローカル開発環境のセットアップ方法です。

## セットアップ

### 必要なもの

- Node.js (v20+)
- npm

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/kou234gh/blog.git
cd blog

# 依存関係をインストール
npm install
```

````

### 環境変数

現在は環境変数は不要です。将来的にCMSを導入する際に必要になります。

### 開発サーバーを起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセス

## セキュリティ

このリポジトリは **Public** で公開されています。

### 注意事項

- APIキーやパスワードは絶対にコミットしない
- 機密情報は`.env`ファイルに記述（`.gitignore`で除外済み）
- 個人情報は含めない

### もし誤ってコミットしてしまったら

1. すぐにそのAPIキー等を無効化する
2. Git履歴から削除する
3. 新しいキーを発行する
````
